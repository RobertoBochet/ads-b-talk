ADS-B

The aircraft language

What is ADS-B

AutomaticDependentSurveillance-Broadcast
is a surveillance system for aircraft traffic

Periodically, aircraft send their own report on the air

What is ADS-B

The report contains

  • Aircraft UID
  • Speed
  • Altitude
  • GPS position
  • Some other information

All this information is sent (also) on 1090MHz

Dump1090

Dump 1090 is an ADS-B (open source) decoder

You can find the source code here

Dump1090

Unfortunately official binaries are not available

And the building script is rather old

Dump1090

So, the fastest way to run Dump1090 is by exploiting docker


        docker run --rm --device /dev/bus/usb -p 8080:8080 jraviles/dump1090

The service GUI is now reachable at http://127.0.0.1:8080
